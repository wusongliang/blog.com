I am using:\
Windows 10\
WSL 2 with Ubuntu20.04\
Nuxt v12.14.12

Had the problem of hot reload not working.\
Solved that by adding

```javascript
 watchers: {
	webpack: {
	  aggregateTimeout: 300,
	  poll: 1000
	}
  }
```

in the `nuxt.config.js` file
